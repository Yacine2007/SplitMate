SplitMate

تطبيق لتقسيم المصاريف بين الأصدقاء والمجموعات. محفظة مشتركة، دعوات آمنة عبر رمز تحقق، واقتراحات لأماكن الخروج.

دعوة آمنة

أضف أصدقاءك عبر رمز تحقق 2FA. أنشئ مساحات مشتركة خاصة.

تقسيم تلقائي

كل مصروف يُقسم بالتساوي بين الأعضاء.

دفع إلكتروني

ادفع حصتك عبر Stripe أو CIB. تتبّع حالة الدفع.

محفظة جماعية

رصيد مشترك للمجموعة مع سجل للحركات.

اكتشف أماكن

مطاعم، سينما، وأنشطة. اربط المصروف بفعالية.

إشعارات

تنبيهات عند إضافة مصروف أو إتمام دفع.

مخطط النشاط

flowchart TD
    Start([بداية]) --> Login[تسجيل الدخول]
    Login --> Has2FA{تفعيل 2FA؟}
    Has2FA -->|نعم| TwoFA[إدخال الرمز]
    Has2FA -->|لا| Dashboard[لوحة التحكم]
    TwoFA --> CodeValid{صحيح؟} -->|نعم| Dashboard
    CodeValid -->|لا| TwoFA
    Dashboard --> Choice{إجراء؟}
    Choice -->|إضافة أصدقاء| Invite[إرسال دعوة + رمز 2FA]
    Invite --> FriendAccept{قبل؟} -->|نعم| SharedSpace[مساحة مشتركة]
    FriendAccept -->|لا| Invite
    SharedSpace --> Dashboard
    Choice -->|مصروف جديد| SelectGroup[اختيار المجموعة]
    SelectGroup --> EnterAmount[إدخال المبلغ]
    EnterAmount --> AutoSplit[تقسيم تلقائي]
    AutoSplit --> NotifyMembers[إشعار الأعضاء]
    NotifyMembers --> ViewShares[مشاهدة الحصص]
    ViewShares --> PayNow{الدفع الآن؟}
    PayNow -->|نعم| OnlinePay[دفع Stripe/CIB]
    OnlinePay --> ConfirmPay{تم التأكيد؟} -->|نعم| UpdateWallet[تحديث المحفظة]
    ConfirmPay -->|لا| Retry[إعادة المحاولة] --> OnlinePay
    PayNow -->|لاحقاً| Pending[قيد الانتظار]
    UpdateWallet --> AllPaid{الكل دفع؟} -->|نعم| Close[إغلاق المصروف]
    AllPaid -->|لا| ViewShares
    Pending --> Close
    Close --> End([نهاية])
    Choice -->|استكشاف| Browse[تصفح الأنشطة]
    Browse --> Filter[تصفية] --> SelectPlace[اختيار مكان] --> SelectGroup
المسار الكامل.
